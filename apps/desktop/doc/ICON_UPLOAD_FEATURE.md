# 密码管理应用 - 自定义图标上传功能

## 功能概述

为密码管理应用新增了自定义图标上传功能，用户可以为每个密码条目上传个性化图标。

## 实现的功能

### 1. 数据库更新
- 在 `passwords` 表中添加了 `icon` 字段（TEXT 类型）
- 更新了数据库初始化脚本，支持现有数据库的自动迁移
- 修改了 `add-password` 和 `update-password` IPC 处理器以支持图标字段

### 2. 前端组件更新

#### AddPasswordModal 组件
- 添加了图标上传区域，支持图片预览
- 实现了文件大小限制（5MB）
- 支持 JPG、PNG、GIF 格式
- 提供了"上传图标"和"移除"按钮

#### EditPasswordModal 组件
- 复用了与 AddPasswordModal 相同的图标上传功能
- 在编辑时正确显示现有图标
- 支持更换和移除图标

#### PasswordList 组件
- 更新了密码列表项的图标显示逻辑
- 优先显示自定义图标，回退到默认的 Globe/Lock 图标

#### PasswordDetail 组件
- 更新了详情页面的图标显示
- 支持显示自定义图标

### 3. 类型定义更新
- 更新了 `Password` 接口，添加了可选的 `icon` 字段
- 确保了类型安全

## 使用方法

### 添加新密码时上传图标
1. 点击"添加密码"按钮
2. 在图标区域点击"上传图标"按钮
3. 选择图片文件（支持 JPG、PNG、GIF，最大 5MB）
4. 预览图标后填写其他信息
5. 保存密码

### 编辑现有密码图标
1. 在密码列表中选择要编辑的密码
2. 点击编辑按钮
3. 在图标区域点击"更换图标"按钮
4. 选择新图片或点击"移除"删除图标
5. 保存更改

## 技术实现细节

### 图标存储
- 图标以 Base64 编码的 Data URL 格式存储在数据库中
- 使用 SQLite 的 TEXT 字段存储图标数据

### 文件上传处理
- 使用 FileReader API 读取本地图片文件
- 实现了文件大小验证和错误处理
- 支持拖拽上传（可扩展）

### 响应式设计
- 图标显示区域在不同组件中保持一致的尺寸
- 支持圆角和溢出隐藏，确保图标显示美观
